## 📌 Overview
The **Career Guide Web Application** is a Django-based platform designed to help students explore career paths, understand required skills, and make informed decisions based on their interests and industry trends.

---

## 🎯 Problem Statement
Traditional career guidance methods:
- ❌ Lack personalization  
- ❌ Provide outdated information  
- ❌ Limited accessibility  

This application solves these issues by offering:
- ✅ Real-time career insights  
- ✅ Structured guidance  
- ✅ Personalized recommendations  

---

## 🚀 Features

### 👨‍🎓 User Features
- 🔐 User Registration & Login  
- 📊 Explore Career Paths  
- 🧠 View Technologies required for each career  
- ❤️ Wishlist (Save preferred careers)  

### 🛠️ Admin Features
- ➕ Add Career Paths  
- ➕ Add Technologies  
- 👥 Manage Users  
- 📊 Dashboard Control  

---

## 🧠 System Architecture
- Follows **Django MTV Architecture (Model-Template-View)**
- Structured database relationships between:
  - Career Paths  
  - Technologies  
  - Users  
  - Wishlist  

---

## 🛠️ Tech Stack

| Category        | Technology                  |
|----------------|---------------------------|
| Backend        | Django (Python)           |
| Frontend       | HTML, CSS, JavaScript     |
| Database       | SQLite / MySQL            |
| Version Control| Git & GitHub              |

---


---

## 📊 Modules Explained

### 🔹 Career Path Module
- Stores career name, image, description  

### 🔹 Technology Module
- Links technologies to careers  
- Includes description & external resources  

### 🔹 Wishlist Module
- Saves user-selected careers
